What is MCP?

Model Context Protocol (MCP) is a standard for building LLM-powered tools. It enables language models to use external tools, resources, and capabilities in a standardized way.

With MCP servers, AI assistants like Claude can access external APIs, retrieve real-time data, and interact with various services while maintaining a consistent interface.

Getting Started

MCP servers can be used with various clients such as Claude Desktop, Cursor, and Windsurf. You can manage MCP servers using the getmcp.sh package manager.

To learn more about MCP and how to contribute your own servers to this registry, visit our GitHub repository.
